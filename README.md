# Pokédex – Python Project (NumPy + Pandas)

A fun data science mini project that works as a Pokédex using Python libraries **Pandas** and **NumPy**.

## Features
- View all Pokémon in a clean table
- Search Pokémon by name
- Filter Pokémon by type
- Display all Legendary Pokémon
- Calculate average stats using NumPy

## Tech Stack
- **Python 3**
- **NumPy** for numerical analysis
- **Pandas** for data handling
